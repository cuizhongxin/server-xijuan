## 效果类型说明

| 效果类型 | 作用方式 | 示例 |
|---------|---------|------|
| **士兵伤害 / 统领士卒伤害** | 最终伤害直加，不经攻防公式 | 伤害＋400 |
| **军队攻击** | 加到 army_att，经攻防公式折算 | 军队攻击＋310 |
| **骑兵/弓兵/步兵 攻击** | 仅限指定兵种，加到 army_att | 骑兵攻击＋300 |
| **骑兵/弓兵 伤害** | 仅限指定兵种，最终伤害直加 | 骑兵伤害＋320 |
| **军队防御** | 加到 army_def，减少受到的伤害 | 军队防御＋350 |
| **步兵防御** | 仅步兵时加到 army_def | 步兵防御＋200 |
| **伤害抵抗** | 直接减少受到的最终伤害 | 伤害抵抗＋180 |
| **士兵生命 +X%** | 按百分比提升部队总生命 | 生命＋12% |
| **军队机动** | 影响出手顺序和追击/撤退 | 机动＋4 |
| **军队闪避** | 提升闪避概率 | 闪避＋12 |
| **统领兵力 +X** | 增加满编士兵上限 | 兵力＋100 |
| **兵法发动概率** | 提升兵法/战法触发率（全兵种） | 兵法概率提升 |
| **骑兵/弓兵兵法概率** | 仅限指定兵种提升兵法触发率 | 骑兵兵法概率提升 |
| **免疫偷袭** | 特殊：免疫偷袭类兵法 | 貂蝉专属 |
